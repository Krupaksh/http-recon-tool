# 🔥 HTTP Recon & Intelligence Tool

A CLI-based Python tool for performing basic HTTP reconnaissance and extracting useful intelligence from web targets.

---

## 🚀 Features

* Sends HTTP requests to target URLs
* Displays response status, time, and content length
* Extracts important headers:

  * Server
  * X-Powered-By
  * Content-Type
  * Set-Cookie
* Clean and structured CLI output
* Basic error handling

---

## 🛠 Tech Stack

* Python 3
* requests library

---

## ▶️ Usage

```bash
python recontool.py https://example.com
```
